常用模运算性质

初等数论：模运算的性质

  1. 加法模运算的性质：(a + b) % m = (a % m + b % m) % m
  2. 减法模运算的性质 ：(a - b) % m = (a % m - b % m) % m
  3. 乘法模运算的性质：(a * a) % m = (a % m * a % m) % m

在大数操作例如幂运算过程中使用取模运算

原码 反码 补码

==正数==

原码=反码=补码

==负数==
原码： 1000 … 0101
反码： 1111 … 1010 原码除了符号位全部按位取反
补码： 1111 … 1011 反码+1

计算机都自动转化成补码进行运算

为什么要用补码运算？ 例如：

1 + -1 = 0
0000 0001 1
1000 0001 -1
1000 0010 -2
//反码：解决了1 + -1 = -2 的问题，但是存在+0 -0
//补码：解决了+0 -0 的问题
